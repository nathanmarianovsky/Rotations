JS Implementation of Vector Rotation

Introduction

Rotation about an axis in three dimensions can be achieved either with the use of a rotation matrix or a unique quaternion mapping. Unlike the matrix method which suffers from "Gimbal Lock", the quaternions do not and offer an increase in performance. To use the web app just provide the necessary information and click submit!
